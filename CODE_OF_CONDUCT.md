# Code of Conduct

This project should be useful and welcoming for creators, developers, designers,
and technical artists.

## Expected behavior

- Be respectful and direct.
- Critique ideas and code, not people.
- Assume different experience levels.
- Keep issues and pull requests focused on the project.
- Share enough context that maintainers and contributors can reproduce problems.

## Unacceptable behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information or credentials.
- Spam, brigading, or unrelated promotion.
- Abusive demands for unpaid support.

## Enforcement

Maintainers may edit, hide, lock, or remove comments and issues that violate
this code of conduct. Repeated or severe violations may lead to a ban from the
project spaces.

If you need to report a conduct issue, contact the maintainers through a private
GitHub channel where available. For security vulnerabilities, use
[SECURITY.md](SECURITY.md) instead.
