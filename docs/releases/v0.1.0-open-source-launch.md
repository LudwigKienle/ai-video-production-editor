# v0.1.0 Open Source Launch Notes

These notes are prepared for the first public source-code launch of AI Video
Production Editor.

Do not push a public `v*` tag until Apple signing and notarization secrets are
configured in GitHub Actions. The release workflow intentionally fails public
tag releases when signing secrets are missing.

## Release Title

AI Video Production Editor v0.1.0 Open Source Launch

## Short Summary

AI Video Production Editor is now open source: a local-first Electron desktop
workspace for AI film production, from script and Director treatment to
storyboard, AI filming, continuity review, editing, and export.

## GitHub Release Body

```md
AI Video Production Editor is now open source.

This release opens the full desktop source code for a local-first AI film
production workspace. The goal is to make AI video feel more like real
production software: project-based, inspectable, extensible, and not locked
behind one hosted prompt form.

## What is included

- Electron + React desktop app source code.
- Project Hub for script-to-filming workflows.
- Director pass and Director treatment flow.
- Concept, storyboard, image generation, and video generation workspaces.
- Studio Agent task queue for resumable production steps.
- Node Space for graph-style creative pipelines.
- Gemini continuity review and re-film queue concepts.
- Timeline, trim, grading, sound, review, request, and export workspaces.
- Provider integrations and adapter surfaces for fast-moving AI media models.
- OpenTimelineIO/FCPXML-style export helpers and VFX handoff utilities.

## Who it is for

- AI filmmakers and video creators who want local project ownership.
- Developers who want to build provider/model adapters.
- Editors and creative technologists exploring AI-assisted production systems.
- Contributors who care about open, inspectable creative software.

## Getting started

```bash
npm install
npm run electron:dev
```

Provider workflows require your own API keys. The app is bring-your-own-key and
provider usage may be billed by the provider.

## LinkedIn Launch Post

```text
I am open-sourcing AI Video Production Editor.

For the last months I have been building it as a local-first desktop workspace
for AI film production: script, Director treatment, storyboard, image
generation, video generation, continuity review, timeline editing, review, and
export in one workflow.

The reason is simple: AI video should not feel like a locked hosted form.

The direction is closer to a Blender-style open core:

- local-first desktop app
- bring your own provider keys
- multi-model workflows
- script -> Director -> storyboard -> AI filming -> review -> export
- open source code contributors can inspect, fork, and improve

I am releasing the source code because I think this kind of production layer
should be available to creators and developers, not only hidden behind expensive
hosted workflows.

It is not finished. That is the point of opening it now.

I would love feedback and contributions around model adapters, UX polish,
continuity workflows, export, sound, docs, and example projects.

GitHub: https://github.com/LudwigKienle/ai-video-production-editor
Trailer: https://youtu.be/-6jo636vRSw?si=v5XSwSIz4WfLtE9z
