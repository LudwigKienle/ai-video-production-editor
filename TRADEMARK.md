# Trademark and Branding Policy

The source code in this repository is licensed under GPL-3.0-or-later. That
license grants rights to copy, modify, and distribute the code under its terms.
It does not grant a separate trademark license to present a fork as the official
AI Video Production Editor project.

## Protected Project Identity

The following project identifiers are reserved for the official project unless
used only for truthful attribution:

- AI Video Production Editor
- AI Video Production Editor logos, icons, and release branding
- Official release claims such as signed, notarized, or maintainer-published
  builds

## Allowed Uses

You may use the project name to:

- State that your work is a fork of AI Video Production Editor.
- Link back to the official project.
- Preserve copyright, license, and attribution notices.
- Describe compatibility with the official project.

## Not Allowed Without Permission

Do not use the project name, logos, or branding in a way that suggests:

- Your fork is the official AI Video Production Editor release.
- The original maintainer endorses your fork.
- You are the original author of this project.
- Your redistributed build is signed, notarized, or published by the original
  maintainer when it is not.

## Recommended Fork Naming

If you publish a fork, use a distinct name and include clear attribution such as:

> Based on AI Video Production Editor by Ludwig Maximillian Kienle.

Official project:
<https://github.com/LudwigKienle/ai-video-production-editor>
