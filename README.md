Official desktop downloads for AI Video Production Editor.

AI Video Production Editor is a local-first desktop studio for AI video
production workflows: planning, generation, media management, editing, review,
and export.

## Download

Download the newest official desktop installers from
[GitHub Releases](https://github.com/LudwigKienle/ai-video-production-editor/releases/latest).

- macOS: signed and notarized `.dmg`
- Windows: `.exe` installer

End users do not need to build, sign, or notarize the app themselves.
